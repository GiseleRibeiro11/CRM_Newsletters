-> Gestão de Newsletters:
Aplicação web desenvolvida com Flask para gestão, agendamento e envio de newsletters.
O sistema permite centralizar campanhas, organizar envios e automatizar tarefas relacionadas à comunicação por e-mail.

-> Descrição do Projeto:
O Gestão de Newsletters é um projeto que simula um sistema real de administração de campanhas de e-mail marketing.
Foi desenvolvido com foco em organização, clareza de código e boas práticas em aplicações web utilizando Flask.
A aplicação conta com rotas organizadas, agendamentos em background e páginas de erro personalizadas, garantindo uma experiência mais profissional e robusta.

-> Funcionalidades:
Autenticação de utilizadores
Dashboard administrativo
Criação e gestão de campanhas de newsletters
Agendamento automático de envios
Envio de e-mails
Upload de ficheiros
Processamento de tarefas em background
Páginas de erro personalizadas (404 e 500)
Interface simples e intuitiva

-> Tecnologias Utilizadas
Python
Flask
Flask-Mail
SQLAlchemy
APScheduler
HTML5
CSS3
SQLite
